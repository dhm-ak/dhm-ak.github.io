01 · Introduction

A short introduction.

I'm a Security Studies student at Leiden University (BSc, graduating 2027), based in The Hague and building toward the intersection of AI governance and frontier-AI security - where technical AI systems meet the institutions that have to govern them (EU AI Act, multilateral security bodies, national safety institutes).

03 · Professional Experience

Mar 2025 - Present · Hybrid
Applied AI Engineer - Adaptive Tutoring System
Independent / Self-employed (Nebula ScoreWorks) The Hague, NL

Built Nebula ScoreWorks, a browser-based analysis engine wrapped by an adaptive AI tutoring layer powered by the Claude API. The system extracts structural features from digital sheet music (key, chords, form, tension) with classical DSP, then hands them to Claude to generate phrase-level explanations tailored to the learner. A dedicated Node.js regression harness treats model behaviour as testable rather than trusted. Wider product / venture story on the Music & Audio page.

  • Integrated the Claude API as an adaptive AI tutoring layer over the deterministic analysis output - phrase-by-phrase explanations, difficulty-scaled to the learner, with structured prompts, output validation, and cost / rate-limit handling.
  • Built a Node.js regression harness that runs the analyser against 909+ pieces from a public reference dataset on every code change, producing diff reports and flagging any accuracy drop - the same pattern I'd apply to evaluating an LLM-based system on a fixed test set.
  • Designed a Viterbi-decoded chord-progression classifier and Krumhansl-Schmuckler key detector - lifted chord-root accuracy from 37 % to 81.85 % on the POP909 benchmark across 6+ measured iterations.
  • Built the analyser as pure client-side JavaScript (no server), and the tutoring UI as a React front-end with a scoped ~3,550-line CSS design system.
  • Tracked down a tricky Web Audio bug silently cutting off the first fifth of a second of every playback; the fix alone lifted a key accuracy metric by 19 percentage points, with a follow-up timing fix adding another 5.
