A short introduction.

This page is the full story of that side of things: the diploma, the sessions, the composition work, and the software I've built for music. The main CV holds the security and software work I'm targeting professionally; this page holds everything music-adjacent that would crowd it.

Everything here is still active. The plan over the next year is to grow Harmonic Conservatory into a real platform and to keep shipping Chord Architect updates alongside the security research at Leiden.

03 · Experience

Where I've built and recorded.

Mar 2025 - Present · Hybrid
Founder & Lead Developer
Harmonic Conservatory · Self-employed The Hague, NL

Founding and developing Harmonic Conservatory, a pre-launch music-technology startup. Its first product, Nebula ScoreWorks, is a website that takes a piece of digital sheet music and breaks it down automatically - showing the chords, the structure, the rises and falls of tension - and writes a plain-English description of what's happening musically.

  • Built the entire analyser in the browser using plain JavaScript - no servers, runs instantly on any computer.
  • Built an automated test system that runs the analyser against 909+ real songs from a public reference dataset, flagging any drop in accuracy on every code change.
  • Designed a classification system that recognises common song structures (verse-chorus-verse patterns and similar) - the same kind of pattern-recognition skill used in OSINT investigation and threat-classification work.
  • Tracked down a tricky audio bug that was silently cutting off the first fifth of a second of every playback. Wrote a clean fix and shipped it.
  • Built an automatic commentary feature: for every musical phrase the system writes a paragraph in plain English explaining what's happening - so beginners can read the analysis without music-theory training.
